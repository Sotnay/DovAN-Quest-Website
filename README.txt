Ouvrir un terminal dans le dossier DovAN-Quest-Website (clic droit) (Après avoir cloner le repo git bien sûr, ou après l'avoir pull)

écrire ces 2 commandes :
- npm install node
- npm install express
- si Tam a décidé d'utiliser bootstrap : npm install bootstrap

puis écrire : node index.js

et dans votre moteur de recherche écrire dans la barre : localhost:53134

Au plaisir de vous revoir!
